## スライド3.5：戦略的背景 (SWOT分析)

Lessons Learned と 2つのチャレンジの架け橋として、現状を維持できない理由を外部・内部の多角的な視点から整理し、戦略の妥当性を補強する。

| 項目 | 内容 |
|------|------|
| **キーメッセージ** | 外部環境の急激な変化と内部課題を照らし合わせると、既存路線の改善だけでは不十分であり、抜本的な変革が必要である。 |
| **表現すべき情報** | ・**強み (S)**：これまでのスキル開発の蓄積、Lessons Learned の基盤と参加意欲<br>・**機会 (O)**：生成AI等の急速な進展、グローバルでの知見共有の加速<br>・**弱み (W)**：リージョン間のサイロ化、データの断絶・共通言語（SFIA）の欠如<br>・**脅威 (T)**：IT人材不足の深刻化、技術変化のスピードアップ |
| **論理構成** | 1. **外部要因（O/T）**：市場は待ってくれない（変革の強制力）<br>2. **内部要因（S/W）**：過去の学びはあるが、仕組みがバラバラで限界（今のままでは勝てない）<br>3. **クロス分析**（別スライド3.6）：弱みを克服しつつグローバルな機会を活かすには、**「基盤の統一」と「グローバルの統合」**が不可欠。 |
| **構成・要素** | ・タイトル：「戦略的背景 (SWOT分析)」<br>・リード文：「外部環境の急激な変化と、内部の現状課題を照らし合わせます。」<br>・軸ラベル：INTERNAL (内部環境) / EXTERNAL (外部環境)<br>・**4象限グリッド**（左上S・右上O・左下W・右下T）。各象限：タイトル（アイコン付き）＋箇条書き。色：S=緑、W=赤、O=青、T=オレンジ。 |
| **文言案** | **S**：これまでのスキル開発の蓄積／Lessons Learnedの基盤と参加意欲<br>**O**：生成AI等の急速な進展／グローバルでの知見共有の加速<br>**W**：リージョン間のサイロ化／データの断絶、共通言語（SFIA）の欠如<br>**T**：IT人材不足の深刻化／技術変化のスピードアップ |
| **補足** | HTML スライド3.5と同一。結論・クロス分析は次のスライド3.6で表示。 |

## スライド3.6：戦略の導出 (クロスSWOT分析)

SWOT 4象限の次に、外部・内部要因から戦略を導く流れと結論を1枚で示す。

| 項目 | 内容 |
|------|------|
| **キーメッセージ** | 外部要因（市場は待ってくれない）と内部要因（仕組みがバラバラで限界）を踏まえ、クロス分析の結論として「基盤の統一」と「グローバルの統合」が不可欠である。 |
| **表現すべき情報** | ・**1. 外部要因 (External)**：市場は待ってくれない（変革の強制力）。機会 (O)・脅威 (T) の箇条書き<br>・**2. 内部要因 (Internal)**：仕組みがバラバラで限界（今のままでは勝てない）。強み (S)・弱み (W) の箇条書き<br>・**3. クロス分析**：強みを活かしつつ弱みを克服し、機会を掴むために「2つのチャレンジ」へ舵を切る<br>・**結論ハイライト**：「基盤の統一」と「グローバルの統合」が不可欠（オレンジピル形状で強調） |
| **構成・要素** | ・タイトル：「戦略の導出 (クロスSWOT分析)」<br>・上段：外部要因ボックス（青系）と内部要因ボックス（オレンジ系）を横並び<br>・中央：下向き矢印<br>・下段：結論ボックス（オレンジ枠）。見出し「3. クロス分析」、本文、その下に結論をピル形状で表示 |
| **文言案** | 外部：「1. 外部要因 (External)」／「市場は待ってくれない（変革の強制力）」＋ O/T の項目<br>内部：「2. 内部要因 (Internal)」／「仕組みがバラバラで限界（今のままでは勝てない）」＋ S/W の項目<br>結論：「強みを活かしつつ弱みを克服し、機会を掴むために『2つのチャレンジ』へ舵を切る」／ **「基盤の統一」と「グローバルの統合」が不可欠** |
| **補足** | HTML スライド3.6と同一。 |
